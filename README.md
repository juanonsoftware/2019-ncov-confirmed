# 2019-ncov-confirmed
JSON data about infected / resolved numbers in the globe

Structure of data
```
[
	{
		date: in format yyyymmdd,
		global_confirmed: number,
		global_new: number,
		global_deaths: number,
		outside_china_confirm: number,
		outside_china_new: number,
		outside_china_deaths: number,
		countries: number,
	},
	{},
	{}
]
```
